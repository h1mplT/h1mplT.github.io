F5隐写

发表于 2017-10-06 | 分类于 知识点 |

简介

F5隐写应用矩阵编码方法，矩阵编码的基本思想是用n个LSB位来表示k比特信息(n>k)。F5提取算法为嵌入算法的逆过程，输入隐藏信息、密钥和载体图像，输出隐写图像。

工具

F5-steganography

linux下载：

git clone https://github.com/matthewgao/F5-steganography

相关题目

http://www.shiyanbar.com/ctf/1938

命令 java Extract ../123456.jpg -p 123456 在F5-steganography文件夹里调用Extract.java解密

搭建Apache+Mysql+PHP环境

发表于 2017-10-06 | 分类于 知识点 |

方法一

Apache

首先根据版本需求从官网下载Apache

下载完成后，根据安装包的提示将其安装

  • 配置环境变量，找到此电脑-属性-高级系统设置-环境变量，找到系统变量path，将Apache安装根目录\bin加入变量

  • 修改配置文件，Apache安装根目录\conf下的httpd.conf，查找ServerRoot，将原地址改为Apache程序的根目录；查找DocumentRoot，为存放网站源文件的目录，将其改为自己的网站源文件路径即可；继续查找，将里面的路径改为和DocumentRoot相同即可

  • 使Apache支持php

配置文件httpd.conf中，

LoadModule php5_module "xxx/php/php5apache2_2.dll"

AddType application/x-httpd-php .php

PHPIniDir "xxx/php/"
  • Apache识别index.php页面

配置文件httpd.conf中，

<IfModule dir_module>
    DirectoryIndex index.html index.php
</IfModule>
  • 浏览器输入http://localhost，如果显示It works!，则配置成功

Mysql

首先根据版本需求从官网下载Mysql

关于安装及配置：

http://blog.csdn.net/luomingjun12315/article/details/50863781

http://www.cnblogs.com/Li-Cheng/articles/4382572.html

PHP

同样是官网下载安装PHP

  • 在根目录中找到php.ini-development的配置文件，将其改名为php.ini

  • 修改php.ini的配置内容，将; extension_dir = "ext"改为extension_dir = "xxx/PHP/ext"

  • 开启相应库功能

    ;extension=php_curl.dll

;extension=php_gd2.dll

;extension=php_mbstring.dll

;extension=php_mysql.dll

;extension=php_xmlrpc.dll

这些是相应的库功能，去掉分号开启功能，其中mysql和mysqli要开启以启用MYSQL数据库支持

虚拟机安装使用

发表于 2017-10-06 | 分类于 知识点 |

  • 工具：VMware

  • 官网下载后，根据安装包提示安装，安装完成后需要用密钥激活，密钥可以多次使用，百度即可

  • 安装完成后，点击创建新的虚拟机，选择典型

这里如果选择安装程序光盘映像文件，则会自动加载iso，打开虚拟机后自动安装系统

建议选择稍后安装系统，选择操作系统与版本

命名与选择安装位置

之后根据自己的需求选择硬盘与配置，其中自定义硬件中新CD/DVD选择使用ISO映像文件

创建完成后，打开虚拟机，按照提示安装系统即可

  • 系统安装完成后，需要安装VMware Tools

选择工具栏 虚拟机-安装VMware Tools

虚拟机中会出现安装包，根据提示安装即可

  • 关于网络配置

选择工具栏 编辑-虚拟网络编辑器

根据需要编辑网络，通常情况默认即可
