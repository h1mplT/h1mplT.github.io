搭建Apache+Mysql+PHP环境

发表于 2017-10-06 | 分类于 知识点 | |

方法一

Apache

首先根据版本需求从官网下载Apache

下载完成后，根据安装包的提示将其安装

  • 配置环境变量，找到此电脑-属性-高级系统设置-环境变量，找到系统变量path，将Apache安装根目录\bin加入变量

  • 修改配置文件，Apache安装根目录\conf下的httpd.conf，查找ServerRoot，将原地址改为Apache程序的根目录；查找DocumentRoot，为存放网站源文件的目录，将其改为自己的网站源文件路径即可；继续查找，将里面的路径改为和DocumentRoot相同即可

  • 使Apache支持php

配置文件httpd.conf中，

LoadModule php5_module "xxx/php/php5apache2_2.dll"

AddType application/x-httpd-php .php

PHPIniDir "xxx/php/"
  • Apache识别index.php页面

配置文件httpd.conf中，

<IfModule dir_module>
    DirectoryIndex index.html index.php
</IfModule>
  • 浏览器输入http://localhost，如果显示It works!，则配置成功

Mysql

首先根据版本需求从官网下载Mysql

关于安装及配置：

http://blog.csdn.net/luomingjun12315/article/details/50863781

http://www.cnblogs.com/Li-Cheng/articles/4382572.html

PHP

同样是官网下载安装PHP

  • 在根目录中找到php.ini-development的配置文件，将其改名为php.ini

  • 修改php.ini的配置内容，将; extension_dir = "ext"改为extension_dir = "xxx/PHP/ext"

  • 开启相应库功能

    ;extension=php_curl.dll

;extension=php_gd2.dll

;extension=php_mbstring.dll

;extension=php_mysql.dll

;extension=php_xmlrpc.dll

这些是相应的库功能，去掉分号开启功能，其中mysql和mysqli要开启以启用MYSQL数据库支持
